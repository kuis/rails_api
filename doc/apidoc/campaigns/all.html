GET /api/v1/campaigns/all
Returns a list of active campaigns. Useful for generating dropdown elements

Returns a list of campaigns sorted by name. Only those campaigns that are accessible for the user will be returned.

Each campaign item have the followign attributes:

  • id: the campaign's ID

  • name: the campaign's name

Supported Formats

json, xml

Errors

400 Bad Request. he server cannot or will not process the request due to something that is perceived to be a client error.
404 Missing
401 Unauthorized access
500 Server crashed for some reason

Examples

GET: /api/v1/campaigns/all.json
[
    {
        "id": 14,
        "name": "ABSOLUT BA FY14"

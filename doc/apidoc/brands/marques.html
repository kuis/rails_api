GET /api/v1/brands/:id/marques
Get a list of marques for a brand

Also see brands#index.

Returns a list of the valid marques for a brand.

Each marque item have the followign attributes:

  • id: the marque's ID

  • name: the marque's name

Supported Formats

json, xml

Errors

404 Missing
401 Unauthorized access
500 Server crashed for some reason

Examples

GET /api/v1/brands/1/marques.json?auth_token=XXXXXYYYYYZZZZZ&company_id=1
[
  {
      "id": "1",
      "name": "Marque #1"
  },
  {
      "id": "2",
      "name": "Marque #2"
  },
  {
      "id": "3",
      "name": "Marque #3"
  },
  ...
]

Params

Param name Description
auth_token
required

User's authorization token returned by login method


Value: Must be String
company_id
required
Value: Must be a number.
id
required

The brand's ID.


Value: Must be a number.

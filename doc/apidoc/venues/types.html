GET /api/v1/venues/types
Get a list of possible venue types

Returns a list of valid venue types that can be used to generate a dropdown when creating a new venue

Supported Formats

json, xml

Errors

404 Missing
401 Unauthorized access
500 Server crashed for some reason

Examples

GET /api/v1/venues/types.json?auth_token=sdFKl0DF9-39tGzWpZ&company_id=1

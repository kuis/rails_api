GET /api/v1/brands
Get a list of brands

Returns a list of brands sorted by name. Only those brands that are accessible for the user will be returned.

Each brand item have the following attributes:

  • id: the brand's ID

  • name: the brand's name

  • active: the brand's status

Supported Formats

json, xml

Errors

400 Bad Request. he server cannot or will not process the request due to something that is perceived to be a client error.
404 Missing
401 Unauthorized access
500 Server crashed for some reason

Examples

GET /api/v1/brands.json
[
  {
      "id": "1",
      "name": "Absolut",
      "active": true
  },
  {
      "id": "2",
      "name": "Jameson LOCALS",
      "active": true
  },
  {
      "id": "3",
      "name": "Ricards",
      "active": true
  },
  ...
]

Params

Param name Description
campaign_id
optional

The brand's ID.


Value:

Must be a number.

GET /api/v1/brands/:id/marques
Get a list of marques for a brand

Also see brands#index.

Returns a list of the valid marques for a brand.

Each marque item have the following attributes:

  • id: the marque's ID

  • name: the marque's name

Supported Formats

json, xml

Errors

400 Bad Request. he server cannot or will not process the request due to something that is perceived to be a client error.
404 Missing
401 Unauthorized access
500 Server crashed for some reason

Examples

GET /api/v1/brands/1/marques.json
[
  {
      "id": "1",
      "name": "Marque #1"
  },
  {
      "id": "2",
      "name": "Marque #2"
  },
  {
      "id": "3",
      "name": "Marque #3"

  ...

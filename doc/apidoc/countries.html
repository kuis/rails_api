GET /api/v1/countries
Get a list of countries

Returns a list of the valid countries in the app. Useful to generate dropdowns.

The list consist on the following attributes:

  • id: the coutry's code

  • name: the coutry's name

Supported Formats

json, xml

Errors

400 Bad Request. he server cannot or will not process the request due to something that is perceived to be a client error.
404 Missing
500 Server crashed for some reason

Examples

GET /api/v1/countries.json
[
  {
      "id": "AD",
      "name": "Andorra"
  },
  {
      "id": "AE",
      "name": "United Arab Emirates"
  },
  {
      "id": "AF",
      "name": "Afghanistan"
  },
  {
      "id": "AG",
      "name": "Antigua and Barbuda"
  },
  ...
]

GET /api/v1/countries/:id/states
Get a list of stages for a country

Also see countries#index.

Returns a list of the valid states for a country.

The list consist on the following attributes:

  • id: the state's code

  • name: the state's name

Supported Formats

json, xml

Errors

400 Bad Request. he server cannot or will not process the request due to something that is perceived to be a client error.
404 Missing
500 Server crashed for some reason

Examples

GET /api/v1/countries/US/states.json
[
  {
      "id": "AK",
      "name": "Alaska"
  },
  {
      "id": "AL",
      "name": "Alabama"
  },
  {
      "id": "AR",
      "name": "Arkansas"
  },
  {
      "id": "AS",
      "name": "American Samoa"
  },
  ...
]

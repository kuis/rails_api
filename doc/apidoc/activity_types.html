Activity types

>>>

GET /api/v1/campaigns/:campaign_id/activity_types
Get a list of contacts for a specific company

Returns a full list of the associated activity types for a campaign

Examples

GET /api/v1/campaigns/1/activity_types
[
    {
        "id": 268,
        "name": "Bar Placement"
    },
    {
        "id": 'attendance',
        "name": "Invitation"
    }
]

Create a new activity

Params

Param name Description
activity
required
Value:

Must be a Hash
activity[activity_type_id]
required

Activity Type ID


Value:

Must be a number.
activity[activity_date]
required

Activity date. Should be in format MM/DD/YYYY.


Value:

Must match regular expression /Ad{1,2}/d{1,2}/d{4}z/.
activity[results_attributes]
optional , nil allowed

A list of activity results with the id and value. Eg: results_attributes: [{id: 1, value:'Some value'}, {id: 2, value: '123'}]


Value:

Must be a list of results [id, value].
activity[company_user_id]
optional , nil allowed

Company user ID


Value:

Must be a number.
activity[campaign_id]
optional , nil allowed

Campaign ID


Value:

Must be a number.

optional , nil allowed

Event ID


Value:

Must be a number.

optional , nil allowed

Venue ID


Value:

Must be a number.

GET /api/v1/events/:event_id/activities
Get a list of activities for an Event or Venue

Returns a full list of the associated activity types for a campaign

Examples

{
  "data": [
    {
      "id": 5135,
      "activity_type_id": 27,
      "activity_type_name": "Jameson BA POS Drop FY15",
      "activity_date": "2015-02-06T02:00:00.000-06:00",
      "company_user_name": "Chris Jaskot"
    }
  ]
}

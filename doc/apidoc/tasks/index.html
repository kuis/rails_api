GET /api/v1/events/:event_id/tasks
Get a list of taks for an Event

GET /api/v1/tasks/teams
Get a list of taks for the user's teams

GET /api/v1/tasks/mine
Get a list of taks for the user

Supported Formats

json, xml

Errors

404 Missing
401 Unauthorized access
500 Server crashed for some reason

Examples

Get a list of active tasks

Params

Param name Description
auth_token
required
Value: Must be String
company_id
required
Value: Must be a number.
event_id
optional

Event ID, required when getting the list of tasks for a event


Value: Must be a number.
status
optional

A list of photo status to filter the results. Options: Active, Inactive


Value: Must be an array of any type
task_status
optional

A list of photo status to filter the results. Options: Late, Complete, Incomplete, Assigned, Unassigned


Value: Must be an array of any type
page
optional

The number of the page, Default: 1


Value: Must be a number.

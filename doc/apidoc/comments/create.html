POST /api/v1/events/:event_id/comments
Create a new comment for an event

Allows to create a comment for an existing event.

Supported Formats

json, xml

Errors

404 Record not found
401 Unauthorized access
500 Server crashed for some reason

Examples

POST /api/v1/events/192/comments.json?auth_token=AJHshslaA.sdd&company_id=1
DATA:
{
  comment: {
    content: 'Text for the first comment'
  }
}

RESPONSE:
{
  {
    "id": 20,
    "content": "Text for the first comment"
    "created_by_id": 1,
    "created_at": "2014-01-07T10:16:39-08:00"
  }
}

Params

Param name Description
auth_token
required

User's authorization token returned by login method


Value: Must be String
company_id
required

One of the allowed company ids returned by the “User companies” API method


Value: Must be a number.
event_id
required

Event ID


Value: Must be a number.
comment
required
Value: Must be a Hash
comment[content]
required

Comment text


Value: Must be String

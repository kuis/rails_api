DELETE /api/v1/events/:event_id/comments/:id
Deletes a comment

Supported Formats

json, xml

Errors

400 Bad Request. he server cannot or will not process the request due to something that is perceived to be a client error.
404 Record not found
401 Unauthorized access
500 Server crashed for some reason

Examples

DELETE /api/v1/events/192/comments/12.json
RESPONSE:
{
  success: true
  info: "The comment was successfully deleted"
  data: { }
}

Params

Param name Description
event_id
required

Event ID


Value:

Must be a number.
id
required

Comment ID


Value:

Must be a number.

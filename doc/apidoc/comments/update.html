PUT /api/v1/events/:event_id/comments/:id
Update a comment

Supported Formats

json, xml

Errors

400 Bad Request. he server cannot or will not process the request due to something that is perceived to be a client error.
404 Record not found
401 Unauthorized access
500 Server crashed for some reason

Examples

POST /api/v1/events/192/comments/12.json
DATA:
{
  comment: {
    content: 'This is the new content for the comment'
  }
}

RESPONSE:
{
  {
    "id": 20,
    "content": "This is the new content for the comment"
    "created_at": "2014-01-07T10:16:39-08:00"
  }
}

Params

Param name Description
event_id
required

Event ID


Value:

Must be a number.
id
required

Comment ID


Value:

Must be a number.
comment
required
Value:

Must be a Hash
comment[content]
optional

Comment text


Value:

Must be String

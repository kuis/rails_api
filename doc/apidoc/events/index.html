GET /api/v1/events
Search for a list of events

Also see User companies.

Returns a list of events filtered by the given params. The results are returned on groups of 30 per request. To obtain the next 30 results provide the <page> param.

All the times and dates are returned on the user's timezone.

Supported Formats

json, xml

Errors

400 Bad Request. he server cannot or will not process the request due to something that is perceived to be a client error.
401 Unauthorized access
404 The requested resource was not found
406 The server cannot return data in the requested format
422 Unprocessable Entity: The change could not be processed because of errors on the data
500 Server crashed for some reason. Possible because of missing required params or wrong parameters

Params

Param name Description
start_date
optional

A date to filter the event list. When provided a start_date without an end_date, the result will only include events that happen on this day. The date should be in the format MM/DD/YYYY.


Value:

Must match regular expression /Ad{1,2}/d{1,2}/d{4}z/.
end_date
optional

A date to filter the event list. This should be provided together with the start_date param and when provided will filter the list with those events that are between that range. The date should be in the format MM/DD/YYYY.


Value:

Must match regular expression /Ad{1,2}/d{1,2}/d{4}z/.
campaign
optional

A list of campaign ids to filter the results

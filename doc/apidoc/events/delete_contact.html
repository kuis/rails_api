DELETE /api/v1/events/:id/contacts
Delete a contact from the event

Supported Formats

json, xml

Errors

400 Bad Request. he server cannot or will not process the request due to something that is perceived to be a client error.
401 Unauthorized access
404 The requested resource was not found
406 The server cannot return data in the requested format
422 Unprocessable Entity: The change could not be processed because of errors on the data
500 Server crashed for some reason. Possible because of missing required params or wrong parameters

Examples

Deleting an user from the event contacts
DELETE: /api/v1/events/8383/contacts.json
DATA:
{
  'contactable_id': 1,
  'contactable_type': 'user'
}

RESPONSE:
{
  'success': true,
  'info': "Contact successfully deleted from event",
  'data': {}
}

Deleting a contact from the event contacts
DELETE: /api/v1/events/8383/contacts.json
DATA:
{
  'contactable_id': 1,
  'contactable_type': 'contact'
}

RESPONSE:
{
  'success': true,
  'info': "Contact successfully deleted from event",
  'data': {}
}

Params

Param name Description
contactable_id
required

The ID of contact/user to be deleted as a contact


Value:

Must be a number.
contactable_type
required

The type of element to be deleted as a contact


Value:

Must be one of: user, contact.

DELETE /api/v1/events/:id/members
Delete an user or team from the event's team

Supported Formats

json, xml

Errors

400 Bad Request. he server cannot or will not process the request due to something that is perceived to be a client error.
401 Unauthorized access
404 The requested resource was not found
406 The server cannot return data in the requested format
422 Unprocessable Entity: The change could not be processed because of errors on the data
500 Server crashed for some reason. Possible because of missing required params or wrong parameters

Examples

Deleting an user from the event members
DELETE: /api/v1/events/8383/members.json
DATA:
{
  'memberable_id': 1,
  'memberable_type': 'user'
}

RESPONSE:
{
  'success': true,
  'info': "Member successfully deleted from event",
  'data': {}
}

Deleting a team from the event members
DELETE: /api/v1/events/8383/members.json
DATA:
{
  'memberable_id': 1,
  'memberable_type': 'team'
}

RESPONSE:
{
  'success': true,
  'info': "Member successfully deleted from event",
  'data': {}
}

Params

Param name Description
memberable_id
required

The ID of team/user to be deleted as a member


Value:

Must be a number.
memberable_type
required

The type of element to be deleted as a member


Value:

Must be one of: user, team.
